# Action Plan 2: Intrinsic Reaction Coordinate (IRC) Implementation

## Overview
Implement Intrinsic Reaction Coordinate (IRC) calculations to follow reaction pathways from transition states to reactants and products. This enables validation of reaction mechanisms and exploration of reaction pathways with steepest descent path following.

```python
    """
    
    def __init__(
        self,
        atoms,

        max_steps=200,
        direction='both',
        predictor='euler',
        corrector='none',

        energy_threshold=1e-6,
        gradient_threshold=1e-4
    ):

        Parameters:
        - atoms: Transition state structure (ASE Atoms)
        - calculator: QME calculator
        - step_size: IRC step size in amu^1/2 * Å
        - max_steps: Maximum number of IRC steps
        - direction: 'forward', 'reverse', or 'both'
        - predictor: 'euler' or 'rk4' (Runge-Kutta 4th order)
        - corrector: 'none', 'gradient', or 'energy'
        - mass_weighted: Use mass-weighted coordinates
        - energy_threshold: Energy convergence threshold
        - gradient_threshold: Gradient convergence threshold
        """
    
    def follow_irc(self) -> Dict[str, Any]:
        """
        Main method to follow IRC path from transition state.
        
        Returns:
            Dictionary containing:
            - forward_path: List of geometries going forward
            - reverse_path: List of geometries going reverse
            - energies_forward: Energies along forward path
            - energies_reverse: Energies along reverse path
            - converged_forward: Boolean convergence status
            - converged_reverse: Boolean convergence status
        """
    
    def _get_initial_direction(self, atoms):
        """Get initial IRC direction from transition vector"""
    
    def _follow_direction(self, atoms, direction_vector, sign=1):
        """Follow IRC in specific direction"""
    
    def _euler_step(self, atoms, gradient, step_size):
        """Euler predictor step"""
    
    def _rk4_step(self, atoms, gradient, step_size):
        """Runge-Kutta 4th order predictor step"""
    
    def _apply_corrector(self, atoms_old, atoms_new, method):
        """Apply corrector step to maintain IRC path"""
    
    def _mass_weight_coordinates(self, coords, masses):
        """Apply mass weighting to coordinates"""
    
    def _check_convergence(self, atoms, energy_change, gradient_norm):

class IRCPathOptimizer:
    """Optimize IRC path points to higher accuracy"""
    
    def __init__(self, path_geometries, calculator):
        """Initialize with IRC path geometries"""
    
    def optimize_path_points(self, fmax=0.01, constrain_to_path=True):
        """Optimize geometries along IRC path"""
    
    def refine_minima(self, fmax=0.001):
        """Refine identified minima to high accuracy"""
    
    def apply_geodesic_refinement(self, use_qme_geodesic=True):
        """
        Apply geodesic interpolation refinement to IRC path points.
        
        Leverages QME's existing geodesic interpolation from qme.reaction.Reaction
        to improve chemical reasonableness of IRC intermediate structures.
        """
        from ..reaction import Reaction
        
        if use_qme_geodesic:
            # Use QME's geodesic interpolation for path refinement
            # Create temporary Reaction object to access geodesic methods
            reaction = Reaction(self.path_geometries[0], self.path_geometries[-1])
            
            # Apply geodesic refinement to intermediate points
            refined_path = reaction._geodesic_interpolation(
                self.path_geometries[0].coords3d,
                self.path_geometries[-1].coords3d,
                len(self.path_geometries)
            )
            
            # Update path geometries with geodesic-refined coordinates
            for i, coords in enumerate(refined_path):
                self.path_geometries[i].set_positions(coords)
        
        return self.path_geometries
```

### 2.1 IRC Integration Algorithms

**Euler Method (simple predictor):**
```python
def euler_step(self, r_current, gradient, step_size, masses):
    """
    Simple Euler integration step
    r_new = r_current - step_size * gradient / sqrt(masses)
    """
    mass_weighted_grad = gradient / np.sqrt(masses.repeat(3))
    r_new = r_current - step_size * mass_weighted_grad
    return r_new
```

### 2.2 Transition Vector Calculation

**Implementation based on frequency analysis:**
```python
def get_transition_vector(self, atoms, calculator):
    """
    Get transition vector from imaginary frequency normal mode.
    This determines the initial IRC direction.
    """
    # Calculate frequencies (requires frequency analysis implementation)
    freq_analysis = FrequencyAnalysis(atoms, calculator)
    frequencies, normal_modes = freq_analysis.calculate()
    
    # Find imaginary frequency (should be only one for proper TS)
    imaginary_idx = np.where(frequencies < 0)[0]
    if len(imaginary_idx) != 1:
        raise ValueError("TS should have exactly one imaginary frequency")
    
    return normal_modes[imaginary_idx[0]]
```
